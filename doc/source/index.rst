Asynchronous MongoDB ORM for Tornado
============================================

.. image:: _static/monguo.jpg

About
-----

Monguo is an asynchronous MongoDB ORM for tornado based on driver `motor <https://github.com/mongodb/motor/>`_.

Install with:

.. code-block:: bash

    $ pip install monguo
